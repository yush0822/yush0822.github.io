1. WithPet — AI 기반 유기동물 입양 플랫폼

React, Spring Boot, MySQL, Docker, FastAPI, Python

프로젝트 소개

유기동물의 입양 확률을 높이기 위해 AI 이미지 생성, 사용자–동물 합성, 보호소 리뷰, AI 추천 챗봇 등을 제공하는 종합 입양 서비스 플랫폼입니다.

내가 맡은 역할

  • React 기반 주요 UI 컴포넌트 개발 (AI 이미지 생성, 채팅 모듈 등)
  • FastAPI 기반 AI 이미지 생성 서버 구축
  • Docker Compose로 백엔드–AI 서버–DB 연결 구조 설정
  • Firebase 스토리지 이미지 업로드/관리 기능 구현
  • AI 챗봇(입양 매칭) 설계 및 프론트엔드 연동

결과 및 성과

  • 입양을 고민하는 사용자에게 “입양 후 모습”을 AI로 시각화하여 서비스 만족도 향상
  • 유기동물 정보 탐색, 추천, 보호소 리뷰, 채팅까지 단일 플랫폼에서 구현

느낀 점

서비스 전체 흐름(기획 → 프론트엔드 → 백엔드 → AI 연동)을 모두 파악하며 기획 단계에서 기술 제약을 고려해야 한다는 점을 깊이 이해하게 되었습니다. 또한 실제 사용자 여정을 생각하면서 기능 배치를 설계하는 과정이 기획자로서의 역량을 강화시켰습니다.

2. 007 Secret Messenger — 암호화 기반 비밀 메신저

서버프로그래밍 강의 프로젝트

프로젝트 소개

텍스트 데이터를 암호화하여 송·수신하는 간단한 비밀 메신저 서비스입니다. 메시지를 안전하게 주고받는 경험을 구현하는 데 집중했습니다.

내가 맡은 역할

  • 전체 기능 흐름 설계(메시지 암호화 → 전송 → 복호화 과정)
  • 클라이언트 UI 구조 설계
  • 서버와의 데이터 처리 방식 정의

결과 및 성과

  • 메시지를 암호화하는 핵심 로직을 성공적으로 구현
  • “데이터 흐름을 이해하는 기획”의 중요성을 실감

느낀 점

간단한 규모의 프로젝트였지만 서버와 클라이언트 간 데이터 흐름을 설계하는 경험을 얻었습니다. 기술적 제약을 이해하는 것이 좋은 기획의 핵심임을 다시 느꼈습니다.
